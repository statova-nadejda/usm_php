### Задание 1.4. Реализация функций

#### `calculateTotalAmount`

Создаём функцию, которая принимает массив транзакций и возвращает общую сумму через `float`. Используем цикл `foreach` для накопления суммы:

```php
function calculateTotalAmount(array $transactions) : float {
    $totalAmount = 0;
    foreach($transactions as $transaction){
        $totalAmount += $transaction["amount"];
    }
    return $totalAmount;
}
```

#### `findTransactionByDescription`

Создаём функцию, которая ищет транзакцию по части строки описания с помощью `str_contains()`. Возвращаем первую найденную транзакцию:

```php
function findTransactionByDescription(string $descriptionPart) {
    global $transactions;
    foreach($transactions as $transaction){
        if(str_contains($transaction["description"], $descriptionPart))
            return $transaction;
    }
}
```

#### `findTransactionById`

Создаём функцию, которая ищет транзакцию по идентификатору с помощью `array_filter()`. После фильтрации применяем `array_values()`, чтобы сбросить исходные ключи массива и корректно получить первый элемент:

```php
function findTransactionById(int $id) : ?array {
    global $transactions;
    $result = array_filter($transactions, function($transaction) use ($id) {
        return $id === $transaction["id"];
    });

    $result = array_values($result);

    return $result[0] ?? null;
}
```

> `array_values()` необходим, потому что `array_filter` сохраняет оригинальные ключи. Без него обращение к `$result[0]` могло бы вернуть `null`, даже если элемент найден (например, транзакция с `id=3` имеет индекс `2`).

#### `daysSinceTransaction`

Создаём функцию, которая вычисляет количество дней между датой транзакции и текущим днём, используя класс `DateTime` и метод `date_diff()`:

```php
function daysSinceTransaction(string $date) : int {
    $transactionDate = new DateTime($date);
    $today = new DateTime();

    return date_diff($transactionDate, $today)->days;
}
```

#### `addTransaction`

### Задание 1.5. Сортировка транзакций

#### Сортировка по дате

Сортируем транзакции по дате по возрастанию с помощью `usort()` и оператора spaceship `<=>`:

```php
function sortArrayByDate() : void {
    global $transactions;

    usort($transactions, function($a, $b) {
        return $a["date"] <=> $b["date"];
    });
}
```

#### Сортировка по сумме (по убыванию)

Сортируем транзакции по полю `amount` в порядке убывания. Для этого меняем операнды местами (`$b <=> $a`):

```php
function sortArrayBySumDescending() : void {
    global $transactions;

    usort($transactions, function($a, $b) {
        return $b["amount"] <=> $a["amount"];
    });
}
```
В методах сортировки мы используем функцию usort(), которая принимает в качестве второго параметра callback-функцию.
Внутри этой функции используется оператор spaceship <=> который возвращает три возможных значения:

-1 - если левый операнд меньше правого
0 - если они равны
1 - если левый операнд больше правого

Этот операнд используется, поскольку `usort()`ожидает от callback отрицательное, нулевое или положительное число, чтобы понять, как расставить два элемента относительно друг друга.
Если результат отрицательный - a идёт раньше b, если положительный - b идёт раньше a.
Поэтому a <=> b даёт сортировку по возрастанию - меньшие даты идут первыми.
А когда мы пишем b <=> a (операнды поменяны местами) - получаем убывание, потому что теперь большие значения возвращают -1 и уходят в начало.

## Контрольные вопросы

### 1. Что такое массивы в PHP?

Массив в PHP — это упорядоченная структура данных, которая позволяет хранить несколько значений под одним именем переменной. Каждый элемент массива состоит из **ключа** и **значения**.

PHP поддерживает два вида массивов:

- **Индексированный (числовой)** — ключи являются целыми числами, начиная с 0 по умолчанию.
- **Ассоциативный** — ключами служат строки, что позволяет обращаться к элементам по смысловому имени.

Массивы в PHP являются динамическими: они могут менять размер во время выполнения программы, а элементы могут содержать значения любых типов, в том числе другие массивы (многомерные массивы).

```php
// Числовой массив
$fruits = ["apple", "banana", "cherry"];

// Ассоциативный массив
$user = [
    "name" => "Alice",
    "age"  => 25,
];

// Многомерный массив
$matrix = [
    [1, 2, 3],
    [4, 5, 6],
];
```

### 2. Каким образом можно создать массив в PHP?

В PHP существует несколько способов создания массивов:

**1. Короткий синтаксис с квадратными скобками (рекомендуемый, PHP 5.4+):**
```php
$numbers = [1, 2, 3, 4, 5];
$person  = ["name" => "Bob", "age" => 30];
```

**2. Конструкция `array()`:**
```php
$numbers = array(1, 2, 3, 4, 5);
$person  = array("name" => "Bob", "age" => 30);
```

**3. Поэлементное добавление:**
```php
$colors = [];
$colors[] = "red";    // автоматически присваивает индекс 0
$colors[] = "green";  // индекс 1
$colors["primary"] = "blue"; // ассоциативный ключ
```

**4. Функции-генераторы:**
```php
$range   = range(1, 10);          // [1, 2, 3, ..., 10]
$filled  = array_fill(0, 5, "x"); // ["x", "x", "x", "x", "x"]
```

### 3. Для чего используется цикл `foreach`?

Цикл `foreach` предназначен для **перебора всех элементов массива или объекта** без необходимости вручную управлять индексом или счётчиком итераций.

Синтаксис:
```php
foreach ($array as $value) {
}

foreach ($array as $key => $value) {
}
```

**Преимущества `foreach`:**
- Читаемость — код становится понятнее по сравнению с `for` или `while`.
- Универсальность — работает как с числовыми, так и с ассоциативными массивами.
- Безопасность — нет риска выйти за пределы массива.

Здесь `foreach` последовательно извлекает каждую транзакцию из массива и обращается к её полю `"amount"` для суммирования.
